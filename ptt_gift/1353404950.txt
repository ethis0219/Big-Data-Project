Jento 的通用工具框架，已經蓋到一半了，功能也開始在成形，,但是開發成本是巨大的。如果通用工具的目標在陽明山，我覺得當時的我們，,頂多頂多只是高雄面向北方，還有非常非常遙遠的距離，才能走到通用工具的成形；,於是我們下了一個決定，我們要一步一步走，不要硬幹。我們得要先從高雄走到台南，,再從台南走到台中，一步一步走到陽明山。,,我記得那幾天，我跟夥伴，把市場上所有的 QR Code 應用都畫在紙上，,然後我們畫出了幾個應用，這幾個應用是我們覺得最適合當做第一步的應用，,其中一個，就是 QR Code 卡片。在經過一些思考後，我們決定 Jento 的第一步，,就是把這個應用做好，讓我們能站穩腳步，逐漸往「通用工具」的目標前進。,,,QR Code 卡片的問題,任何產品，都是要解決人類的問題。當時市面上有一些 QR Code 卡片，,一個大剌剌的 QR Code 圖，就這樣印在卡片上面，收卡人掃描之後，,就可以看到一些寫卡者準備的內容。我覺得這個很棒，因為除了手寫的文字，,在這個數位的時代，我們還可以使用屬於我們的方式，去給收卡者，,屬於我們這個時代的感動。,,但是我買不下去，因為這類產品有個嚴重的問題：「QR Code卡片很醜」，,我怎麼樣都不會想要送這樣的一張卡片給我女朋友，醜死了。送卡片的時候，,我想要挑選一張適合對方的卡片，與對方氣息相符合的卡片，而這類的 QR Code 卡片，,我想不到有任何適合的朋友，是屬於這種又粗糙、又神祕、又不漂亮、又礙眼的氣息。,,我們想要做的更好。於是我們想到了，為什麼消費者不能買他屬意的卡片，然後依然有,QR Code卡片的功能？於是我們把 QR Code 做成了貼紙，讓消費者可以自由搭配卡片。,,當時，對我自己而言，我才會想買。因為我的卡片不會醜，,我可以送給那個總是古靈精怪的女孩，一張外星人卡片，,我用 QR Code 貼紙對她說：「妳真的很奇怪，古怪，但我會一直當妳的古怪朋友的」，,然後送她一首 Dreamer。這才是我要的產品。,,而且我想起，我也送過前女友類似的東西，但是它不是一張 QR Code 貼紙，,它是一張醜陋至極的影印紙剪裁成的
